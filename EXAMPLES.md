## Tips for Best Results

### ‚úÖ DO:
- Be specific about core features
- Mention the main action (track, manage, generate, etc.)
- Include 1-2 key features
- Keep it under 15 words

### ‚ùå DON'T:
- Ask questions
- Specify technologies
- Include "please" or "can you"
- List 10+ features

### Examples:

**Good:**
```
"Expense tracker with categories and monthly reports"
"Recipe manager with ingredient search"
"Todo list with due dates and priorities"
```

**Too Vague:**
```
"An app for managing stuff"
"Something to track things"
"A helpful tool"
```

**Too Complex:**
```
"A fully featured enterprise CRM with AI predictions, blockchain integration, real-time collaboration, advanced analytics, and mobile apps"
```

- Edit `frontend/src/App.jsx` for UI changes
- Modify `backend/server.js` for API logic
- Update `supabase/schema.sql` for database changes
- Customize `frontend/src/index.css` for styling
